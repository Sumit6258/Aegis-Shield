🛡️ Aegis Shield – Air Defence Interceptor Simulator

🚀 Live Demo: https://sumit6258.github.io/Aegis-Shield/

🌌 Overview

Aegis Shield is a highly realistic, browser-based 3D Air Defence System Simulator that models real-world radar detection, tracking, and missile interception systems.

It blends physics, mathematics, and interactive gameplay to simulate how modern air defence systems detect and neutralize aerial threats such as drones, cruise missiles, ballistic missiles, and fighter jets.

This project is designed as both:

🎮 An interactive simulation/game

🧠 A deep engineering visualization tool

⚡ Key Features
🛰️ Real-Time Radar System

Rotating radar sweep (military-style HUD)

Detection of multiple aerial threats

Range-based tracking with dynamic blips

Real-time target updates

✈️ Enemy Threat Simulation

Simulates different types of aerial threats with realistic behavior:

🛸 Drones – slow, erratic movement

🚀 Cruise Missiles – low altitude, high precision

☄️ Ballistic Missiles – parabolic trajectories

🛩️ Fighter Jets – high-speed evasive motion

Each object includes:

Velocity

Altitude

Trajectory patterns

Detection characteristics

🎯 Interceptor System
